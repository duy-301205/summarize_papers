📘 Summarize Papers - Frontend

Đây là ứng dụng frontend được xây dựng bằng React + Vite nhằm hỗ trợ người dùng tóm tắt các bài báo khoa học thông qua giao diện web.

🛠️ Công nghệ sử dụng

- React

- Vite

- JavaScript

- Axios / Fetch API

- CSS / TailwindCSS

⚙️ Yêu cầu hệ thống

Trước khi chạy project, cần cài đặt:

- Node.js (>= 18.x)

- npm (được cài sẵn cùng Node.js)

Kiểm tra Node.js đã được cài đặt hay chưa:

- node -v

Kiểm tra npm:

- npm -v

Nếu chưa cài đặt Node.js, tải tại:

https://nodejs.org

⚙️ Cài đặt project
1. Clone repository
git clone https://github.com/ten-repo.git
2. Di chuyển vào thư mục chứa file src project
cd summarize_papers
3. Cài đặt dependencies
npm install
▶️ Chạy project ở môi trường phát triển
npm run dev

Sau đó truy cập:

http://localhost:5173
